# FinalShell Clone

一个基于 FinalShell 3.8.3 静态分析的全功能 SSH 终端管理工具。

## 功能特性

### 核心功能
- **SSH 终端**: JediTerm 终端模拟器，支持完整的终端功能
- **SFTP 文件管理**: 双面板文件浏览器，支持拖拽传输
- **FTP 文件传输**: FTP/FTPS 支持，双面板文件管理
- **系统监控**: CPU、内存、磁盘、网络实时监控
- **端口转发**: 本地/远程/动态端口转发管理
- **代理支持**: HTTP、SOCKS4、SOCKS5、跳板机
- **RDP 远程桌面**: 通过 SSH 隧道连接 Windows 远程桌面
- **VNC 远程桌面**: VNC 协议支持，可通过 SSH 隧道
- **Telnet 连接**: Telnet 协议实现，支持自动登录
- **Zmodem 传输**: rz/sz 文件传输协议
- **批量执行**: 多服务器并发命令执行
- **进程管理**: 远程进程查看/终止
- **文件权限**: chmod/chown 可视化编辑
- **文件搜索**: 按文件名/内容/大小/时间搜索
- **内置编辑器**: 远程文件编辑，多编码支持
- **网络工具**: Ping/端口扫描/Traceroute/DNS查询
- **脚本引擎**: JavaScript自动化脚本
- **热键管理**: 自定义快捷键配置
- **布局保存**: 窗口布局保存/恢复
- **国际化**: 中英文多语言支持
- **插件系统**: 可扩展插件架构
- **UI主题**: Light/Dark/Dracula/Monokai 主题切换
- **设置中心**: 完整的配置管理界面
- **SSH密钥管理**: 密钥生成/导入/导出
- **路由追踪/WHOIS**: 网络诊断工具
- **速度测试**: 网络带宽测试

### 高级功能
- **终端主题**: 6种内置主题 (Default, Dark, Monokai, Dracula, Solarized)
- **快捷命令**: 自定义命令列表，快速执行
- **命令历史**: 命令记录和搜索
- **数据同步**: 本地导入/导出，WebDAV 同步支持

## 系统要求

- **Java**: JDK 8 或更高版本
- **操作系统**: Windows / macOS / Linux
- **内存**: 最低 256MB，推荐 512MB+

## 快速开始

### 编译项目

```bash
# 克隆项目
git clone <repository-url>
cd finalshell-clone

# 编译打包
mvn clean package
```

### 运行应用

**Windows:**
```bash
scripts\start.bat
```

**Linux/macOS:**
```bash
chmod +x scripts/start.sh
./scripts/start.sh
```

**或直接运行:**
```bash
java -jar target/finalshell-clone-1.0.0-SNAPSHOT-jar-with-dependencies.jar
```

## 使用说明

### 创建 SSH 连接

1. 点击左侧工具栏的 **+** 按钮
2. 填写连接信息：名称、主机、端口、用户名
3. 选择认证方式：密码或私钥
4. (可选) 配置代理设置
5. 点击 **确定** 保存

### 文件传输

1. 连接到 SSH 服务器
2. 点击 **SFTP** 按钮打开文件面板
3. 左侧为本地文件，右侧为远程文件
4. 拖拽或双击进行传输

### 端口转发

1. 连接到 SSH 服务器
2. 点击 **端口转发** 按钮
3. 添加转发规则：
   - **本地转发**: 将本地端口转发到远程
   - **远程转发**: 将远程端口转发到本地
   - **动态转发**: SOCKS5 代理

### 数据备份

1. 菜单 → 工具 → 数据同步
2. 选择导出位置
3. 点击 **导出数据**
